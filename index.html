Bienvenidos a Tree of Savior Castellano

Esta página ha sido creada con el fin de compartir nuestro proyecto Tree of Savior Castellano. Con este proyecto queremos ofrecer a los hispanohablantes una traducción precisa y de alta calidad en un castellano neutro. Podéis visitar el GitHub del proyecto aquí.
